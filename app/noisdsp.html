VALKYRIE

Description

VALKYRIE is an experimental distortion effect unit that seeks to achive the effect which had changed our perception of music forever. The DISTORTION. This experimental plugin is loaded with 5 distinct custom made funtions to achive the sounds which range from light boosts to complete chaos with just a change in knob.


Algorithmns

VALKYRIE is loaded with 5 distinct distortion algorithmns:

  • ARRA: A very mild saturation function that can be used as a booster when cranked high enough.

  • SIGX: A darker sounding distortion function that envokes the qualities of UK style amps.

  • HTAN: A diode distortion with crisp sound and controls.

  • FCLP: A heavy fuzz style sound with asymetrical controls.

  • DNRC: A heavy rectifier sound that can become chaotic very fast.


Controls

VALKYRIE consistes of a number of parameters for finetuning effect outpot.

  • Distortion: This parameter selects the distortion algorithmn to be used.
  • Gain: This parameter controls the saturation of the distortion effect. (This control is unneeded for the FCLP and DNRC algorithmns).
  • Asymmetry: This control effects the asyymetical clipping of the signal. (Only applies to FCLP and DNRC algorithmns).
  • Mid Frequency: By default VALKYRIE applies a 2nd order High Pass Filter on the signal. This parameter controls the corner frequency of the low pass filter.
  • Low: This parameter controls the gain of the Low Shelf Filter.
  • High: This parameter controls the gain of the High Shelf Filter.
